linux 命令行下的计算器bc

在分析.MOV文件格式时需要计算地址，于是google到了bc神器

bc - An arbitrary precision calculator language

bc是一个任意精度的计算器语言，我的乖乖，正是神器啊！！！

DESCRIPTION
bc is a language that supports arbitrary precision numbers with interactive execution of statements. There are some simi‐ larities in the syntax to the C programming language. A standard math library is available by command line option. If requested, the math library is defined before processing any files. bc starts by processing code from all the files listed on the command line in the order listed. After all files have been processed, bc reads from the standard input. All code is executed as it is read. (If a file contains a command to halt the processor, bc will never read from the standard input.)

1 基本计算 加减乘除 

edgar@debian:/$ echo '1+1' | bc
2
edgar@debian:/$ echo '3-2' | bc
1
edgar@debian:/$ echo '2 * 5'| bc
10
edgar@debian:/$ echo '8 / 3'| bc     <--- 像C语言
2
edgar@debian:/$ echo '8 % 3'| bc
2
edgar@debian:/$ echo '4 % 3'| bc
1

2 使用上一次的计算结果 

edgar@debian:/$ echo '11 * 3' | bc
33
edgar@debian:/$ echo 'last + 2' | bc
2
edgar@debian:/$ echo '11*3; last+2' | bc
33
35
edgar@debian:/$ echo '11*3; .+2' | bc
33
35

3 指定精度 默认小数点后显示0位 

edgar@debian:/$ echo '8/3' | bc
2
edgar@debian:/$ echo 'scale=1; 8/3' | bc
2.6
edgar@debian:/$ echo 'scale=10; 8/3' | bc
2.6666666666
edgar@debian:/$ echo 'scale=10; sqrt(3)' | bc
1.7320508075

4 进制之间相互转换 默认输入输出都是十进制 

edgar@debian:/$ echo 'obase=2; 12' | bc   十进制到二进制
1100
edgar@debian:/$ echo 'obase=8; 12' | bc   十进制到八进制
14
edgar@debian:/$ echo 'obase=16; 12' | bc  十进制到十六进制
C
edgar@debian:/$ echo 'obase=2; ibase=16; 12' | bc  十六进制到二进制
10010

5 复杂运算需要使用括号 

edgar@debian:/$ echo 'obase=10; ibase=10; (3^3)^3' | bc
19683
edgar@debian:/$ echo 'obase=10; ibase=10; 3^3^3' | bc
7625597484987

6 十六进制的都使用大写 

edgar@debian:/$ echo 'obase=10; ibase=16; ffff' | bc
0
edgar@debian:/$ echo 'obase=10; ibase=16; FFFF' | bc
65535
edgar@debian:/$ echo 'obase=10; ibase=16; 1a' | bc
(standard_in) 1: syntax error
edgar@debian:/$ echo 'obase=10; ibase=16; 1A' | bc
26

7 我就想要小数点有10个数字出现 

edgar@debian:/$ echo 'scale=10; 1.25*9' | bc
11.25
edgar@debian:/$ echo 'scale=10; (1.25*9)/1' | bc
11.2500000000
